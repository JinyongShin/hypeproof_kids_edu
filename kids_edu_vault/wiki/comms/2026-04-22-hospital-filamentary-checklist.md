## 병원 측 전달

### 집합/환경
- [ ] 40명 동시 집합 허가 여부 (소아암 병동 면역저하 환자 고려)
- [ ] 대상 연령대 확정 (현재 8-12세 가정)
- [ ] Wi-Fi 40대 동시 접속 부하테스트 — 병원 IT팀 협의
- [ ] 핫스팟 백업 기기 반입 허가
- [ ] 보호자 동반 여부

### 촬영 (필라멘트리 TJ 피드백)
- [ ] 촬영 가능 구역 — 강당 내부 한정? 입장/이동 동선까지?
- [ ] 노출 금지 항목 — 병실, 병동 표지판, 이름표, 의료 장비, 차트/모니터 화면 일괄 금지?
- [ ] 병원 로고, 내부 시설, 의료진 모습 포함 기준
- [ ] 촬영 주체 — HypeProof, 동아일보, 병원 홍보팀 동일 기준?
- [ ] 장비 반입 — 카메라, 짐벌, 삼각대, 무선마이크, 조명, 멀티탭, 인원 제한
- [ ] 미디어 동의서 주체 — 병원 처리 vs HypeProof 처리

### 영상물 활용 (TJ 피드백)
- [ ] 행사 스케치, SNS 숏폼, 홈페이지/랜딩, 제안서/소개자료까지 자유 게시?
- [ ] 구분 필요 시 — 어디까지 자유, 어디부터 사전 확인?
- [ ] 편집 표현 가이드라인 — 질병/치료 과도 암시, 취약성 강조 연출, 과한 감동 서사 금지?
- [ ] 병원명과 함께 사용 가능한 표현 수위
- [ ] 레퍼런스 활용 — 병원명, 관계자 코멘트, 현장 반응을 사례 자산으로 사용 가능?
