## 블록별 운영자 스크립트

### 블록 1: 지휘관의 선언 (13:30~13:45)
> "오늘 여러분은 지휘관이에요! AI에게 명령해서 나만의 캐릭터를 만들 거예요."
> 
> 시연: "별을 모으는 게임 만들어줘" (좋은 명령) vs "뭐 만들어줘" (나쁜 명령)
> 
> "구체적으로 말할수록 AI가 더 잘 만들어줘요!"

### 블록 2: 아바타 소환 (13:45~14:05)
> "이제 네 캐릭터를 소환해볼까요? 어떤 캐릭터를 만들고 싶어요?"
> 
> 프롬프트 예시 버튼이 화면에 나타남. 아이가 선택하거나 자유 입력.
> 
> ⚠️ 타이핑이 어려운 아이: 운영자가 대신 입력
> ⚠️ 15분 경과: "다음엔 어떤 세계에 살까요?"로 자연스럽게 다음 블록 유도

### 블록 3: 세계 구축 (14:05~14:25)
> "캐릭터가 어디에 살고 싶어요? 바다? 우주? 꽃 숲?"
> 
> 카드 완성 → 갤러리 자동 업데이트

### 블록 4: 마스터 개발자 (14:35~15:00)
> "이제 진짜 개발자처럼 카드를 꾸며볼까요?"
> 
> 완성된 카드에 QR 생성 → 사원증 인쇄/화면 표시

### 블록 5: 런칭쇼 (15:00~15:20)
> 갤러리 슬라이드쇼 (/gallery) 대형 스크린에 투사
> 각 아이 카드 소개: "OOO 개발자님의 [캐릭터 이름]!"

## 에러 대응

| 상황 | 메시지 | 운영자 행동 |
|---|---|---|
| AI 응답 지연 (15초+) | "마법사가 생각 중이에요..." | "조금만 기다려요!" |
| AI 오류 | "마법사가 쉬고 있어요" | "다시 한 번 말해볼까?" → 재시도 |
| 콘텐츠 필터 | AI가 대안 제시 | 자연스럽게 다른 방향 유도 |
| Wi-Fi 끊김 | 접속 끊김 | 핫스팟 백업 전환 |
| 아이 이탈 | — | 현재 블록만으로 성취 가능하니 걱정 말 것 |

## 운영자 체크리스트

### 사전 (13:00)
- [ ] 기기 40대 충전 확인
- [ ] 전 기기 동일 URL 사전 입력
- [ ] Wi-Fi 부하 테스트 완료
- [ ] 핫스팟 백업 2대 준비
- [ ] 대형 스크린 미러링 확인
- [ ] 갤러리 페이지 (/gallery) 정상 동작 확인
- [ ] QR 생성 테스트 완료

### 당일 주의사항
- 아이 8명당 운영자 1명 배치 (총 5명)
- 타이핑 지원: 어린 아이는 운영자가 대신 입력
- 기술 용어 절대 사용 금지
- 아이가 만든 결과물은 반드시 칭찬
- 이탈 아이: 걱정 말 것, 다시 참여하면 현재 블록부터
