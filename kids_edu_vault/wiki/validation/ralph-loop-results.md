## 발견 및 수정한 버그

### 🔴 Critical: GLM 성공 시 card/done 이벤트 미발생 (FIXED)
- **파일**: `genai_runner.py` `generate_card()`
- **원인**: 카드 JSON 추출과 `done` 이벤트가 `except Exception as glm_err` 블록 안에만 있어서, GLM이 정상 응답하면 frontend가 영원히 "카드 만드는 중..." 상태로 멈춤
- **수정**: card 추출 + done 이벤트를 try/except 밖으로 이동 (성공/실패 공통 경로)
- **영향**: 이 버그가 있으면 파일럿 전체가 불가능했음

## 성능

| 항목 | 시간 |
|------|------|
| 프롬프트 → 텍스트 응답 | ~5-15초 |
| 이미지 생성 | ~10-15초 |
| 전체 카드 생성 | ~25-35초 |

⚠️ **주의**: 파일럿(40명 동시)에서는 응답 시간이 길어질 수 있음. 부하 테스트 필수.

## 아직 테스트 못 한 항목
- 매우 긴 입력 (500자+)
- 영어 입력
- 특수문자 only
- "게임 시작해줘" (캐릭터 없이)
- "이전 카드 수정해줘" (이전 없이)
- 연속 빠른 입력 (스팸)
- 15초+ 대기
- 여러 세션 동시 생성
- 세션 삭제 후 복구
- QR 생성 확인
