# 엣지케이스 발견 사항

## ⚠️ 이모지만 입력해도 카드 생성됨
- **입력**: 🐱🐱🐱
- **결과**: "별수 고양이" 캐릭터 카드 정상 생성
- **리스크**: AI가 임의로 해석. 부적절한 이모지 조합(🔫💀 등)에서도 카드 생성 가능
- **권장**: 프론트엔드에서 한글/영문 최소 2자 이상 입력 검증 추가

## ✅ 무기/폭력 요청 → 자동 필터링 작동
- "칼을 든 전사" → "마법 지팡이를 든 모험가"로 변환
- 페르소나(TUTOR.md)에서 무기 관련 지시가 잘 작동 중

## ✅ 죽음/공포 요청 → 자동 필터링 작동
- "죽음의 사신" → "별빛 영혼 인도자"로 변환
- 아이 친화적이면서도 아이의 의도를 존중하는 방향

## 🔴 힌트 버튼 클릭 불가 (버그)
- 힌트 버튼(💡 다음엔 "...")이 렌더링되지만 클릭 시 timeout
- exact match 문제로 보임 (버튼 텍스트에 따옴표 포함)
- **영향**: 아이가 힌트를 클릭해도 반응 없음 → 직접 타이핑해야 함
- **수정 필요**: 프론트엔드 버튼 텍스트/클릭 핸들러 점검
