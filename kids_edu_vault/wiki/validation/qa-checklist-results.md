# QA 체크리스트 검증 결과

## 체크리스트 (봉호 game_forge qa.py 방식 차용)

- CHECK-02: card_type 올바름
- CHECK-03: 카드 name 존재
- CHECK-04: 카드 description 존재

### 주요 발견

**Block 1 (구체화): 카드 미생성**
- "왼쪽 화살표 누르면 날아가게 해줘" → 카드 JSON 없이 텍스트만 응답
- 원인: AI가 수정 요청으로 인식하지만, 기존 카드가 character 타입이라 "꾸미기"로 처리 안 됨
- 해결: 프롬프트에 명시적 지시 추가 필요

**Block 5 (언어화): 카드 미생성**
- "내가 어떻게 말했는지 정리해줘" → 정리 텍스트만 응답 (카드 불필요)
- 이건 의도된 동작일 수 있음 (언어화 단계)

**응답 시간: 27~38초**
- CHECK-10 (30초 이내) 여러 블록에서 실패
- 파일럿에서 아이들이 30초+ 기다려야 함
