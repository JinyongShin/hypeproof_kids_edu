# 바이브 코딩 (Vibe Coding)

## 개념

자연어로 AI에게 지시해 코드를 생성하는 방식. 코드 문법이나 디버깅 지식 없이도 **의도와 감각(vibe)으로 소프트웨어를 만드는** 패러다임. Andrej Karpathy가 명명.

## HypeProof 파일럿에서의 의미

본 파일럿에서 아이들이 하는 행위의 본질 — "기사 캐릭터를 은빛 갑옷을 입고 왼쪽 눈에 흉터가 있게 만들어줘"처럼 **구체적인 묘사로 AI를 지휘**하는 것. 코딩 지식 없이 의도를 실현.

브리핑 문서([[2026-04-20-meeting-briefing]])에서 핵심 교육 패러다임으로 언급.
