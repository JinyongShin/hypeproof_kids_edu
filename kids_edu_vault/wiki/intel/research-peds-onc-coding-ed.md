# Research: 소아암 환아 대상 코딩·AI 교육 선례

## Overview

2026-05-05 HypeProof Kids Edu 파일럿(국립암센터 강당, 8–12세 40명, 90분 6블록 AI 코딩 교육)에 적용 가능한 국내외 선례를 조사. **결론: 우리 파일럿 형식(소아암 환아 대상 단일 회차 대규모 AI 코딩 이벤트)의 직접 선례는 peer-reviewed 문헌과 공개 프로그램 기록에서 확인되지 않음.** 인접 영역의 증거는 존재하며, 이를 조합하면 우리의 교수법·운영 프로토콜·감염관리·피로도 대응을 근거 기반으로 설계 가능.

## Key Findings

### 교수법 (Pedagogy)
- **Code.org 기반 코딩이 인지 손상 아동에게 실현 가능하고 효과적**이라는 peer-reviewed 증거가 존재. 단, 11세션 x 5-6주 규모에서 (Source: [[case-sickle-cell-coding-study]]).
- **단회성 90분 이벤트로 EF 전이는 주장 불가**. 교수 목표는 **경험·정서·노출**로 설정해야 정직 (Source: [[case-sickle-cell-coding-study]], [[case-oep-socioecological-program]]).
- 소아암 환아의 공통 인지 제약(주의·처리속도·피로)에 대한 **표준 수업 편의**는 잘 정립되어 있음: 스크라이브, 연장 시간, 조용한 공간, 자주 체크인, 예고 없는 지명 금지 (Source: [[case-stjude-educational-challenges]]).
- 우리 설계의 4가지 병동 맞춤 변경 — [[no-debug-philosophy]], [[ai-persona-workflows]], [[single-html-runtime]], [[combat-vs-cooperative-framing]] — 은 **모두 St. Jude 편의 권고와 구조적으로 일치**.
- **"schooling matters" 메시지 + 치료로부터의 주의 분산**이 부모가 평가한 가치의 핵심. EF 전이 같은 과장 주장보다 방어적이며 진실된 프레이밍 (Source: [[case-oep-socioecological-program]]).

### 기술 스택 / 전달 모드
- 병원 교육학 프레임워크는 **3가지 전달 모드** 확립: bedside / async micro-learning / synchronous remote. **대규모 인퍼슨 이벤트는 없는 4번째 모드** (Source: [[case-hospital-pedagogy-framework]]).
- Starlight Children's Foundation이 **tech-as-play 패러다임**을 대규모로 확립 — 82% 병원 파트너가 불안·스트레스 감소 보고. 단 bedside 1:1 중심 (Source: [[case-starlight-therapeutic-gaming]]).
- Pediatric Blood & Cancer 2025 "Techquity" 프레임워크가 **AI + 몰입형 tech를 소아·AYA 종양학의 정당한 개입 영역으로 확립**. 우리 파일럿은 이 프레임에 정확히 들어맞음 (Source: [[case-techquity-pediatric-oncology]]).
- 호주 OEP의 WebEx 디지털 파트는 **"원하는 가정은 많았으나 실제 사용은 0"** — 기술 제공만으로는 충분하지 않음. 운영 설계가 진짜 병목 (Source: [[case-oep-socioecological-program]]).

### 감염관리 (파일럿 당일 필수)
- **손 위생이 단일 최강 통제**. 시각 자료 + **운영자 구두 프롬프트** 복합이 가장 효과적 (Source: [[case-pediatric-onc-infection-control]]).
- **모든 참석자 사전 증상 문진 + 발열 체크** 의무. 증상자 입장 불가.
- **ANC <100 호중구감소** 환아는 대규모 실내 모임 contraindication — 의료진과 컷오프 사전 합의.
- ≥12 ACH 환기 기준 (HSCT 병실 표준). 강당 HVAC 확인 + HEPA 백업 결정.
- 1인 1기기 원칙, 사용 전후 70% 에탄올 와이프.
- 보수적 표준으로 **전원 수술마스크** 권장.

### 세션 길이 / 피로 관리
- 미국 기준 **병원 내 수업 시간은 보통 하루 1시간** (Source: St. Jude Together 인용 검색 결과).
- 우리의 **90분 6블록 (블록당 ≤15분)** 설계는 표준 하루치를 넘지 않으면서 자연스러운 **재휴식·이탈·재합류 공간**을 제공. 실증 증거 기반 설계와 합치.
- 피로·처리속도 결손 대응은 **입력 부담 최소화 (AI가 대부분 타이핑) + 자주 짧은 성취 지점**이 표준 권고.

### 심리사회적 고려 (Psychosocial)
- 학업 연속성은 **standard of care** — "잘 훈련된 팀 구성원" 필요. 단순 자원봉사 ≠ 충분 (Source: [[case-academic-continuity-peds-onc]]).
- 우리 튜터는 **사전 오리엔테이션 필수** — [[pilot-operator-guide]] 반영.
- **또래 교육 세션 + 교사 워크숍**이 재진입 프로그램에서 입증된 성과 (놀림 감소, 태도 개선) → 우리 콘텐츠의 **학교 배포 후속**이 저비용 확장 기회.
- 전투 서사는 외부 증거 기반으로도 피해야 함 — Starlight을 비롯한 주요 기관은 협력·돌봄 프레이밍이 표준 (Source: [[case-starlight-therapeutic-gaming]] → [[combat-vs-cooperative-framing]] 뒷받침).

## Key Entities

- **St. Jude Children's Research Hospital** — 소아암 교육·임상 표준 확립의 글로벌 중심. "Together by St. Jude"는 환아·가족 대상 공식 가이드.
- **Starlight Children's Foundation** — 824개 병원 파트너, 2024년 300만 환아 임팩트. Therapeutic Gaming Program 운영.
- **Memorial Sloan Kettering Pediatrics** — M401 학교 프로그램 운영(세부는 비공개).
- **UNICA4EU Project** — 유럽 소아암 AI 혁신 프로젝트.

## Key Concepts

- **Techquity** — 소아·AYA 종양학에서 AI + 몰입형 tech로 형평성 격차를 줄이는 개입 프레임워크 (Puthenpura 2025).
- **Hospital Pedagogy** — 3가지 전달 모드 (bedside / async / sync remote). 5가지 핵심 메커니즘.
- **Socioecological Framework (Bronfenbrenner)** — OEP의 근거. 환아 개인이 아니라 가족·병원·학교 **시스템**을 타깃.
- **Therapeutic Gaming** — 치료적 의도로 게임·VR을 스트레스·불안 감소에 사용하는 임상 패러다임.

## Contradictions

- **EF 전이 주장 가능 여부**: [[case-sickle-cell-coding-study]]는 11세션 집약 투여 결과 EF 전이 입증. 우리 90분 단회성 이벤트는 이를 주장할 증거 없음. 정직한 목표는 **노출·경험·정서**.
- **대규모 group vs bedside**: 기존 문헌은 거의 모두 bedside 1:1 또는 소규모. 40명 동시 참여 precedent가 peds-onc에서 확인되지 않음. 우리는 상대적 **novel implementation** — 잠재적 publishable 영역.

## Next Actions (for the pilot)

1. **의료진 합의**: ANC 컷오프·참석 가능성 사전 문진 양식 — 국립암센터 담당의와 4/20 전 확정.
2. **운영 프로토콜 작성**: 손 위생·문진·마스크·기기 와이프를 [[pilot-operator-guide]]에 런북 형태로 편입.
3. **튜터 사전 오리엔테이션 커리큘럼** 작성 — 단순 기술 자원봉사 ≠ 충분. 소아암 심리사회적 기본 + 감염관리 + 우리 페다고지 제약.
4. **성과 지표 재조정**: EF 전이 주장 폐기. 대신 ① 완주율 ② 자기효능감 self-report ③ 부모 만족도 ④ 치명 장애 0 — OKR Q2 ([[okr-q2-jy]])와 재정렬.
5. **논문화 가능성 검토**: 국내 문헌 기준 최초 사례이므로 IRB 승인 + 사전 동의 구조를 갖추면 publishable.
