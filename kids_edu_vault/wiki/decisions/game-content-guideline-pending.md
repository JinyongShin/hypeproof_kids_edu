# ADR: 소아암 환아용 게임 콘텐츠 가이드라인 — 의료 전문가 확인 전 보류

## 결정 요약

Claude 프롬프트에 소아암 환아 대상 게임 생성 제약 조건을 추가하는 작업을 **국립암센터 또는 담당 의료진의 실제 확인을 받기 전까지 보류**한다.

## 배경

딥리서치 결과, 아래 게임 요소들이 소아암 환자 치료 철학과 불일치할 가능성이 제기되었다.

- 적(敵) 캐릭터 등장 및 전투 요소
- 체력/HP 감소 메커니즘
- 죽음·실패 연출

이 우려를 바탕으로 `src/backend/main.py`의 게임 생성 프롬프트 주입부에 제약 조건을 하드코딩하는 방안이 검토되었으나, 아래 이유로 보류한다.

## 보류 이유

> [!warning] 의료 전문가 확인 필요
> 딥리서치는 기존 사례 기반 **추정**이며, 실제 임상 현장의 치료 철학과 다를 수 있다. 잘못된 가이드라인을 코드에 고정하면 의료 전문가 피드백 반영 시 재작업 비용이 발생한다.

1. 딥리서치 결과는 일반 문헌 기반 추정 — 국립암센터 해당 병동의 실제 방침과 다를 수 있음.
2. 제약 조건을 조기에 코드에 고정할 경우, 전문가 의견 수렴 후 변경 시 프롬프트 로직 전체 재작업 필요.
3. 파일럿(2026-05-05) 준비 일정상 잘못된 가이드라인 반영이 더 위험함.
