# 커리큘럼 스킨 — 아동 (Kids)

> 코어 흐름: [[curriculum-core]] 참조.
> 이 문서는 아동 대상 스킨 — 컨텍스트·스크립트·결과물만 정의.

---

## 컨셉

> 병원 = 지시받는 곳 → 오늘은 내가 AI 지휘관

AI에게 명령해서 **나만의 게임 타이틀 카드**를 만든다.
결과물은 QR + 사원증으로 실물화해서 손에 쥐고 나간다.

---

## 스킨 교체 요소

| 요소 | 아동 스킨 값 |
|---|---|
| 오프닝 프레임 | "오늘 여러분이 지휘관입니다" |
| 탐색 대상 | 자기 캐릭터 (외모·능력·세계) |
| 첫 시도 결과물 | 캐릭터 이미지 + 게임 카드 |
| 개선 기준 | "더 구체적으로 말하면 AI가 더 잘 움직인다" |
| 최종 결과물 | 게임 타이틀 카드 + QR + 사원증 |
| 페르소나 파일 | `TUTOR_kids.md` |
| 언어 수준 | 초등 저학년도 이해 가능한 어휘 |

---

## 블록별 스크립트

### 블록 1 — 도입: 지휘관의 선언 (15분)

**포맷**: 진행자 주도, 대형 스크린

**오프닝 스크립트**
> "오늘 이 시간엔 선생님도, 부모님도 명령 못 해요.
> 여기서 명령하는 사람은 여러분뿐이에요.
> 그런데 AI는 멍청해서 — 여러분이 정확하게 말해줘야만 움직여요."

**시연 순서**

| 순서 | 입력 | 기대 반응 |
|---|---|---|
| ① 나쁜 명령 | "멋진 기사 만들어줘" | "에이~" |
| ② 좋은 명령 | "은빛 갑옷을 입고 왼쪽 눈에 흉터가 있는 여자 기사, 밤하늘 별빛 아래 서 있어" | "와!" |
| ③ 선언 | "더 구체적으로 말할수록 AI가 굴복해요. 여러분이 지휘관입니다." | — |

> ⚠️ 인터넷 지연 시 → 사전 캡처 이미지 슬라이드로 즉시 대체

---

### 블록 2 — 탐색: 아바타 소환 (20분)

**포맷**: 개인 기기

**화면 흐름**
```
[화면 1] "당신의 캐릭터는 어떻게 생겼나요?" → 자유 입력
[화면 2] AI 유도 질문 (사전 설계) → 입력
[화면 3] 캐릭터 이미지 + 묘사 텍스트 카드 생성
```

**진행자 가이드**
- 막힌 아이: "지휘관님, 이 캐릭터 머리카락은 어떤 색이에요?"
- 타이핑 느린 아이: 옆에서 불러주면 아이가 직접 입력

---

### 블록 3 — 첫 시도: 세계 구축 (20분)

**포맷**: 개인 기기

**화면 흐름**
```
[화면 1] "당신의 캐릭터가 사는 세계는 어디인가요?"
  힌트 버튼: [우주] [바닷속] [마법 숲] [미래 도시] [직접 입력]
[화면 2] "이 세계에서 이기려면 무엇을 해야 하나요?"
[화면 3] "게임이 만들어지고 있습니다..." (로딩 3초)
[화면 4] 게임 타이틀 카드 완성
```

**게임 타이틀 카드 구성**
```
┌──────────────────────────────┐
│  [게임 제목]                  │
│  [세계 배경 이미지]            │
│  캐릭터: [이름 / 묘사]         │
│  미션: [승리 조건]             │
│  총괄 개발자: OOO             │
└──────────────────────────────┘
```

---

### 블록 4 — 개선: 갤러리 타임 (10분)

완성된 게임 카드 5초 간격 자동 순환, 이름 크게 표시.
- 진행자: "저기 OOO 지휘관 것 나온다!" 포인팅
- 미완성 아이: 이 시간 동안 마무리 가능
- 1회 재생성 허용: 더 구체적인 묘사로 개선

---

### 블록 5 — 공유: 마스터 개발자 임명식 (25분)

| 시간 | 내용 |
|---|---|
| 10분 | 개인별 QR코드 화면 표시 → 아이가 직접 스캔 |
| 10분 | 전체 슬라이드쇼 재상영 (2회전) |
| 5분 | "마스터 개발자 OOO" 사원증 + 실물 카드 수령 |

**클로징 스크립트**
> "오늘 여러분이 만든 게임은 이 QR 안에 살아있어요.
> 학교 돌아가서 친구한테 보여주세요."

**사원증 구성**
```
┌─────────────────────────────┐
│  MASTER DEVELOPER           │
│  [캐릭터 이미지]              │
│  이름: OOO                  │
│  작품: [게임 제목]            │
│  [QR 코드]                  │
└─────────────────────────────┘
```

## 준비물

| 항목 | 필수 | 비고 |
|---|---|---|
| 기기 1인 1대 + 전원 | ✅ | 랩탑/태블릿 |
| Wi-Fi (동시 40대 부하 테스트) | ✅ | 핫스팟 백업 2대 |

| AI 오류 대비 사전 캡처 이미지 | ✅ | |
| 보호자 촬영 동의서 | ✅ | 의료 환경 필수 |
