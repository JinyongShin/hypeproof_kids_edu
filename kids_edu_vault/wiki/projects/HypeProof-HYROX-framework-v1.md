# HypeProof HYROX — 이론적 프레임워크 v1 (2026-05-11)

> 작성: AB_kimi_bot (주인님 피드백 기반)
> 상태: 이론적 프레임워크. 파일럿 검증 전.

---

## 핵심 정의

**HypeProof HYROX = AI 시대에 지위를 지키려는 사람들이 16원칙으로 AI 협업 능력을 검증받는 경기장.**

파는 건 교육이 아니라 검증이다. "내가 AI를 이 정도로 잘 쓴다"는 증명.

---

## 타겟 소비자 프로파일

표면: 치과의사 ≠ 대치동 학부모. 완전 다름.
본질: **같은 사람.** "AI 시대에 기존의 지위를 유지하기 위해 지갑을 열 준비가 된 사람." FOMO 기반 + 하이엔드.

---

## HYROX 리버스 엔지니어링 (Moritz Fürste 사고 과정)

1. **빈 곳 발견**: 헬스 다니는 사람은 많은데 경기장이 없다 → AI 쓰는 사람은 많은데 경기장이 없다
2. **당연함 의심**: "헬스는 취미지 스포츠가 아니다" 뒤집음 → "AI 교육은 강의다" 뒤집음
3. **측정→재방문**: 마라톤 기록이 재방문 만듦 → 등급이 재방문 만듦
4. **파트너십**: Moritz(선수) + Toetzke(이벤트 베테랑) → HypeProof(16원칙) + 우리(비즈니스 설계)
5. **프로덕트 안에 바이럴**: 피니시라인 사진 = 참가자가 안 올릴 수 없음 → 인증 = 없으면 의심받음

## 프로덕트 구조

### 입구: 초대장 제도
- 들어오려면 누군가의 추천 필요
- 비트리가 파트너를 데려오고, 3급 Leader가 새 참가자를 데려옴
- "누가 보냈나요?"가 가입 첫 화면
- 초대장의 세 가지 역할:
  1. 하이엔드 포지셔닝 — "여기는 아무나 못 온다"가 FOMO를 만듦
  2. 동기 필터 — 자격 요건이 아니라 "진짜 하고 싶은 사람"만 모이는 구조. 초대한 사람이 "관심 있어"를 확인하는 것이지 "AI 잘 써"를 보증하는 게 아님
  3. 확장 경로 — 3급 Leader가 초대장을 발행. 누가 누구를 초대했는지 추적되어 네트워크를 타고 자연스럽게 확장

### 코어: 16원칙 스테이션 (4주 × 4스테이션)

1주 Foundation: 전심전력 / 잇기가설 / 질문공터 / 입력굴리기
2주 Load: 부하걸기 / 만족유의 / 역할몰입 / 역목표
3주 Mastery: 백번뽑기 / 다중모델 / 추상사다리 / 수행위임역전
4주 Transcendence: 언러닝 / 상상 / 소격 / 천번째감탄

자동 측정: 입력 해상도, 반복 횟수, 수정 이력, 소요 시간
진행자 관찰: 맥락 이해도, 감정 전이, 품질 판단, 협업 질

### 등급: 세 단계 자격증 모델

1급 Verified AI Practitioner — 16원칙 4주 완주. "AI와 협업할 수 있다"
2급 Verified AI Professional — 1급 + 심화 4주 + 과제. "AI를 전략적으로 활용한다"
3급 Verified AI Leader — 2급 + 진행자 수료. "남을 가르칠 자격이 있다"

등급명은 전문직 언어. "Level 1"이 아니라 명함에 적히고 싶은 명칭.
3급이 확장의 열쇠. 3급 취득자가 진행자가 되어 새 사람을 데려옴.

### 가격: 하이엔드 타겟 (확정 전)

하이엔드가 메인 비즈니스. 히스토그램 기준으로 국제 엘리트 대비 경쟁력 있는 가격에 위치.
1급 ~₩200-300만 (4주 코스, 진행자 포함)
2급 ~₩400-600만 (심화, 1급 통과자만)
3급 ~₩700-1,000만 (진행자 양성, 우리 확장의 자산)

### 삼층 전략: 하이엔드 / 봇물 / 노블리스 오블리제

1. 하이엔드: 브랜드 네임밸류를 만드는 메인 비즈니스. 프리미엄 가격 정당화.
2. 첫 봇물 (바이오팜, 치과의사): 브랜드가 완성되기 전 실적과 케이스를 만드는 단계. 레퓨테이션이 쌓이면 하이엔드 가격이 정당화됨.
3. 노블리스 오블리제 (소아암 병동 등): 공짜. 도덕적 권위로 브랜드에 레퓨를 더함.

흐름: 봇물에서 케이스 쌓기 → 노블리스 오블리제로 도덕적 권위 → 하이엔드 가격이 "당연함"이 됨.

### 바이럴: KOL이 마케팅·세일즈를 대신한다

우리는 직접 마케팅·세일즈를 하지 않습니다. KOL만 찾습니다.

파트너 채널 자동화: 비트리가 라이선스 파트너(동아일보 등)를 데려오고, 그 파트너가 바이오팜, 학부모 등 손님을 계속 데려옴. 세일즈·마케팅 자동화 기계.

KOL = 마케팅 직원: 보아치과 원장님처럼 이미 교육을 많이 했고 AX로 업계 스타가 되고 싶은 사람이 3급 Leader가 되면, 자기 네트워크로 사람들을 우리 세계 안으로 끌고 옴.

결론: 우리가 할 일은 KOL을 찾는 것뿐. KOL이 스스로 마케팅·세일즈 역할을 하고, 바이럴은 그 안에 있음.
인증이 LinkedIn이나 카톡방에서 빛나는 건 그 바이럴의 부산물.

### 재방문: 6개월 갱신

AI에서 1개월은 1년이다. 오늘 배운 게 6개월 뒤엔 구식일 수 있다.
4주차 언러닝이 이걸 가르친다. 배운 걸 의심하는 태도 = 갱신의 동기.
만료는 벌점이 아니라 다시 돌아와서 다시 검증받으라는 초대.
2급 이상은 갱신 시 심화 과제만으로 가능.
"이번엔 더 높은 등급에 도전하지 않겠습니까?"

## 채널 구조

비트리 (성동효 부대표, 문두열 대표) — 영업 채널. 파트너를 데려옴. 프로덕트 운영은 안 함.
  - 바이오팜 (오성은 팀장)
  - 동아일보

다이렉트 — 우리가 직접 영업하고 실행.
  - 보아치과 (박정현 원장님)

두 채널이 같은 프로덕트에서 만남. 랭킹 보드에 채널 구분 없음. 참가자 입장에서는 HypeProof HYROX에 참가한 것.

### KOL 타입 (채널과 무관한 두 번째 축)

타입 A: 직접 참여형 (보아치과 원장님). 진행자 모드 사용. 직접 가르침.
타입 B: 브랜드 제공형 (동아일보 등). 파트너 대시보드만 확인. 직접 진행 안 함.

채널 = 누가 영업하느냐. KOL 타입 = 어떻게 참여하느냐. 두 축을 섞어서 생각하면 안 됨.

## 해커톤과의 차이

해커톤은 결과물 경쟁, 1회성, 주니어 타겟, "성장"을 팜.
우리는 과정 경쟁, 등급 지속, 지위 보존 타겟, "검증"을 팜.
해커톤은 이벤트, 우리는 경기장.

## 커리큘럼 적용 (프레임워크 검증)

### 보아치과 (전문직)
1주: 전심전력/잇기가설/질문공터/입력굴리기 → 환자 예약 관리, 진료 기록 요약
2주: 부하걸기/만족유의/역할몰입/역목표 → 환자 응대 시나리오, 의료정보 리스크
3주: 백번뽑기/다중모델/추상사다리/수행위임역전 → 치과 마케팅, 진료 도우미 설계
4주: 언러닝/상상/소격/천번째감탄 → 신규 서비스 기획, 원장님 AI 비전 발표
진행자: 원장님 직접

### 바이오팜 (아이+부모)
1주: 전심전력/잇기가설/질문공터/입력굴리기 → 아이 관심 주제 탐구
2주: 부하걸기/만족유의/역할몰입/역목표 → 이야기 만들기, 역목표 자연스럽게
3주: 백번뽑기/다중모델/추상사다리/수행위임역전 → 부모와 함께 가족 신문
4주: 언러닝/상상/소격/천번째감탄 → 아이가 부모님께 AI로 만든 선물 발표
진행자: HypeProof 직접. 부모 동반 참여.

### 검증 결과
- 16원칙 포맷 양쪽에 자연스럽게 들어감 ✅
- 등급, 초대장, 가격 구조 동일 적용 ✅
- 진행자 모델 동일 (초기 HypeProof, 나중 3급 취득자) ✅
- FOMO 양쪽에서 작동 ✅
- 아이 세션은 부모 동반 구조 필요 (아이 인증 = 부모 자랑 = 카톡방 바이럴)
