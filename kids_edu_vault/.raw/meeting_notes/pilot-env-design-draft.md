## 아이들이 경험하는 것

**접속 URL: `https://hypeproof-ai.xyz`**

```
1. 링크 접속 (태블릿 / 노트북 브라우저)
2. Google 계정으로 로그인
3. 브라우저에서 VS Code 열림 + 게임 스타터 파일 자동 탑재
4. AI 채팅창에 말을 걸면 Gemini가 게임을 함께 만들어줌
5. 브라우저에서 바로 게임 실행 → 즉각적인 시각 피드백
```

설치 없음. 코딩 몰라도 됨. AI가 도와줌.

## 기술 구성 (JY 담당)

| 역할 | 도구 | 비고 |
|---|---|---|
| 브라우저 VS Code | code-server | MIT 오픈소스 |
| Google 로그인 | oauth2-proxy | 어린이 계정 화이트리스트 설정 |
| AI 채팅 (Cursor처럼) | Cline extension + Gemini 2.5 Flash | 아이들이 API Key 설정 불필요 |
| HTTPS 자동 인증서 | Caddy | Let's Encrypt 자동 발급 |
| 실습 콘텐츠 | sans-kids-school-2025 자산 재활용 | 게임 템플릿 + 워크플로우 |

**AI 튜터 모드:** 소아암 병동 특성상 **빠른 구현 모드** 권장
- 아이들이 원하는 걸 말하면 AI가 바로 만들어줌
- 기다리는 시간 최소화, 성취감 극대화
