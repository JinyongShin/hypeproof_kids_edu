전체 방향
==========================================================

상품: AI 크리에이터 워크샵
대상: 소아암병동 8-12세 환아 40명
시간: 2시간 (13:30-15:30)
목표: 아이가 AI와 대화해서 자기만의 창작물을 만들고 가져가는 경험

실행환경
강당 확보 + 장비 반입 제한 없음 → Track A (웹 기반) 주력 준비.
다만 40명 전원 랩탑 확보가 관건. 부족분은 태블릿/폰 병행 (Track B 백업).
- Track A (주력): 웹 기반 채팅 + 프리뷰, 랩탑/태블릿 사용
- Track B (백업): 텔레그램 챗봇 (OpenClaw 기반), 폰 사용
- 커리큘럼은 하나, 실행환경만 스위칭

운영 구조
40명을 1명이 진행할 수 없음. 조 편성 필수.
- 8조 x 5명 또는 10조 x 4명
- 조별 퍼실리테이터 1명 배치 (팀원 + 자원봉사자)
- 메인 진행자 1명 (무대/스크린)

검증 대상은 기술이 아니라 상품
- 아이가 2시간 안에 완성작 1개를 만들 수 있는가
- 아이와 보호자가 재참여 의향이 있는가
- 병원이 후속 워크샵을 요청하는가
- 동아일보가 후속 제안/기사로 이어지는가
- 기술 장애 0건은 전제 조건이지 성공 지표가 아님

고려 사항
- 동아 내빈 + 병원장 인터뷰 → 행사 앞뒤에 인사/소개/클로징 시간 필요
- 기사 플레이 가능 → TJ 촬영과 동아 취재진 동선 사전 협의
- 콘센트 17개로 40대 충전은 부족 → 완충 상태로 가져가거나 멀티탭 준비
- 피로도 제약 → 중간 휴식 필수, 블록 길이 20분 이내 권장
- 사전 등록 출입 → 팀원 + 자원봉사자 전원 명단 사전 제출
